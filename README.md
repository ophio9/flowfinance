# Budget Manager

A modern, responsive Budget Manager application built with HTML, CSS, and JavaScript. The app helps users track income, expenses, budgets, savings goals, and recurring transactions while storing data locally in the browser for a seamless offline experience.

## Features

### Income Tracking

* Add and manage multiple income sources
* Record amount, date, and description
* View total income summaries

### Expense Tracking

* Log daily expenses
* Organize expenses by category
* Edit or delete transactions

### Budget Management

* Create monthly budget limits
* Track spending against budgets
* Receive alerts when approaching budget limits

### Savings Goals

* Create savings targets
* Monitor progress toward goals
* Track remaining amounts needed

### Recurring Transactions

* Set recurring income and expenses
* Automatically generate scheduled transactions
* Reduce manual data entry

### Dashboard & Analytics

* View current balance
* Monitor income vs. expenses
* Analyze spending patterns
* Visual charts and summaries

### Notifications

* Budget limit warnings
* Savings goal milestones
* Upcoming recurring transactions

### Persistent Local Storage

* Automatically saves data in the browser
* No database or backend required
* Data remains available after page refreshes or browser restarts

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage API

## Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/budget-manager.git
```

2. Navigate to the project folder:

```bash
cd budget-manager
```

3. Open `index.html` in your browser.

No additional dependencies or installation steps are required.
